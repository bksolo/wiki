# Marp

Markdown-basiertes Slide-Deck-Format. Erlaubt es, Präsentationen direkt aus Markdown zu generieren. [[Obsidian]] hat ein Plugin dafür.

## Relevanz

Im [[llm-wiki-pattern]] ein mögliches Output-Format: Das LLM kann Antworten auf Queries als Marp-Slides rendern, die in [[obsidian]] direkt angezeigt werden. Slides werden in `output/slides/` gespeichert.
